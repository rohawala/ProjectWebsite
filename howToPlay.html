How To Play:

For this VR project, the team agreed on creating a basic game using the Unity 3D game engine. Since the team had little knowledge of Unity and game development as a whole, with thought hard about what can be developed with the skills we have. We end up creating a simple collection game, called The Toilet Paper Shortage or The TP Shortage for short.
